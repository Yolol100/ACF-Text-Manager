# Security Policy

## Supported versions

Security fixes are provided for the latest released version of ACF Page Text Manager.

| Version | Supported |
|---|---|
| 2.5.x | Yes |
| Older versions | No |

## Reporting a vulnerability

Please report suspected security issues privately instead of opening a public issue.

Contact: security@webactueel.nl

Include as much detail as possible:

- affected version;
- WordPress, PHP, and ACF version;
- user role required to trigger the issue;
- exact steps to reproduce;
- whether the issue requires administrator access, editor access, subscriber access, or logged-out access.

We aim to acknowledge valid reports promptly and will coordinate a fix before public disclosure where appropriate.

## Security expectations

ACF Page Text Manager is an administrator/editor workflow plugin for managing text content, ACF fields, SEO metadata, and selected media metadata. It should be installed only for trusted WordPress sites and reviewed on staging before bulk imports or media filename changes.

The plugin does not phone home, does not load remote assets, and does not include telemetry.
